NutriScan usa la cámara y la IA de Gemini para identificar alimentos, estimar porciones y avisarte si algo no encaja con tu dieta o tus alergias.

Se guarda solo en este dispositivo (localStorage) y se usa para llamar directo a la API de Google desde el navegador. Consíguela gratis en aistudio.google.com/apikey.
